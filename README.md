# 🌫 AQI Prediction using Machine Learning

This project predicts the **Air Quality Index (AQI)** using Machine Learning based on pollution levels.

---

## 🚀 Features
- Predict AQI using input pollutants
- Classify AQI (Good, Moderate, Poor, etc.)
- Graph visualization of pollution levels
- Feature importance analysis

---

## 📊 Input Parameters
- CO (Carbon Monoxide)
- NO2 (Nitrogen Dioxide)
- SO2 (Sulfur Dioxide)
- O3 (Ozone)
- PM2.5 (Fine Particles)
- PM10 (Coarse Particles)

---

## 🎯 Output
- AQI Value
- AQI Category
- Graph Visualization

---

## 🧠 Machine Learning Model
- Random Forest Regressor

### Why Random Forest?
- Handles complex data
- Reduces overfitting
- High accuracy
